# 🌦️ Weather Data Analytics Dashboard

An interactive **Weather Data Analytics Dashboard** built using **R Shiny** to analyze, visualize, and generate insights from weather datasets. The application provides an intuitive interface for exploring weather patterns, performing statistical analysis, and generating reports through interactive visualizations.

---

## 📌 Features

- 📊 Interactive weather data visualization
- 🧹 Data cleaning and preprocessing
- 📈 Exploratory Data Analysis (EDA)
- 📉 Statistical summary and correlation analysis
- 📄 Automated report generation
- 🎨 Interactive plots using **ggplot2**
- ⚡ Modular R Shiny application structure

---

## 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/your-username/weather-data-analytics-dashboard.git
```

### Navigate to the project

```bash
cd weather-data-analytics-dashboard
```

### Install required packages

```R
install.packages(c(
  "shiny",
  "ggplot2",
  "dplyr",
  "tidyr",
  "DT",
  "plotly",
  "rmarkdown"
))
```

### Run the application

```R
shiny::runApp()
```
